AstroBob Documentation

Memory-Powered Agent Toolkit for IBM Bob. Give your AI agents persistent semantic, episodic, and procedural memory via AstraDB.

рҹҺҜ What is AstroBob?
AstroBob is a memory system that gives IBM Bob (and other AI agents) the ability to remember, learn, and improve across sessions. Instead of forgetting everything between conversations, agents can store facts, recall events, and apply learned procedures.

Key Features

рҹ§  Three Memory Types

Semantic - Stable facts and knowledge
Episodic - Events and experiences
Procedural - Reusable how-to procedures

рҹ”Ң MCP Integration

5 powerful tools for Bob:
remember, recall, reflect, forget, audit_trail
Seamless integration via Model Context Protocol

рҹ“Ҡ Smart Ranking

Weighted scoring system:
55% rerank + 15% importance + 15% recency + 10% success + 5% staleness
Always surfaces the most relevant memories

рҹҺ“ Self-Improving

Bob can reflect on experiences and write its own skills
Procedural memories вҶ’ Bob SKILL.md files
Creates a self-improvement feedback loop

рҹҡҖ Production Ready

Built on AstraDB Serverless
Hybrid search with reranking
70%+ test coverage, fully typed

рҹӨқ Multi-Agent Support

WxO integration templates
Shared memory across agents
Coordination patterns included

Quick Example

# Install AstroBob
uv tool install astrobob

# Initialize in your project
cd ~/projects/my-project
astrobob init
astrobob astra setup

# Store a memory
astrobob memory remember \
  --type procedural \
  --content "To deploy: 1) Run tests, 2) Build, 3) Deploy to staging, 4) Verify, 5) Deploy to prod" \
  --importance 5 \
  --tags deployment,procedure

# Recall it later
astrobob memory recall "how to deploy"

# Export as Bob skill
astrobob skills sync

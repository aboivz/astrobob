Installation

Get AstroBob installed and running in under 5 minutes. Choose your preferred installation method below.

📋 Prerequisites:
• Python 3.12 or higher
• AstraDB Serverless account (free tier available)
• IBM Bob trial or subscription (optional, for MCP integration)

Installation Methods

Method 1: Using uv (Recommended)

uv is the fastest and most modern Python package installer. Perfect for quick installations.

# Install uv if you don't have it
curl -LsSf https://astral.sh/uv/install.sh | sh

# Install AstroBob as a tool
uv tool install astrobob

# Verify installation
astrobob --version
✅ Recommended: This method installs AstroBob in an isolated environment and makes the astrobob command globally available.

Method 2: Using pipx

pipx installs Python applications in isolated environments, similar to uv tool.

# Install pipx if you don't have it
python -m pip install --user pipx
python -m pipx ensurepath

# Install AstroBob
pipx install astrobob

# Verify installation
astrobob --version

Method 3: From Source (Development)

Install from the GitHub repository for the latest development version or to contribute.

# Clone the repository
git clone https://github.com/aboivz/astrobob.git
cd astrobob

# Install with uv (recommended)
uv sync

# Or install with pip
pip install -e .

# Verify installation
astrobob --version

Method 4: Direct pip Install

Standard pip installation from GitHub.

# Install directly from GitHub
pip install git+https://github.com/aboivz/astrobob.git

# Verify installation
astrobob --version

Verifying Installation

After installation, verify that AstroBob is working correctly:

# Check version
astrobob --version

# View help
astrobob --help

# List available commands
astrobob

# Run health check (requires configuration)
astrobob doctor

Expected output:

$ astrobob --version
astrobob version 0.1.0

$ astrobob --help
Usage: astrobob [OPTIONS] COMMAND [ARGS]...

  AstroBob - Memory-Powered Agent Toolkit for IBM Bob

Commands:
  astra    Manage AstraDB collections
  doctor   Run health checks
  init     Initialize AstroBob in current project
  mcp      Start MCP server
  memory   Manage memories
  skills   Manage Bob skills

Troubleshooting Installation

Python Version Issues

⚠️ Error: Python 3.12 or higher is required

Solution: Install Python 3.12+ from python.org or use a version manager like pyenv:

# Using pyenv
pyenv install 3.12
pyenv global 3.12

Permission Errors

⚠️ Error: Permission denied

Solution: Use --user flag or a virtual environment:

# Install for current user only
pip install --user astrobob

# Or use a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install astrobob

Command Not Found

⚠️ Error: astrobob: command not found

Solution: Ensure the installation directory is in your PATH:

# For uv tool
export PATH="$HOME/.local/bin:$PATH"

# For pipx
pipx ensurepath

# Then restart your terminal

Dependency Conflicts

⚠️ Error: Dependency resolution failed

Solution: Use an isolated environment:

# Recommended: Use uv tool or pipx (already isolated)
uv tool install astrobob

# Or create a fresh virtual environment
python -m venv astrobob-env
source astrobob-env/bin/activate
pip install astrobob

Platform-Specific Notes

Windows

  • Use PowerShell or Command Prompt
  • Activate virtual environments with venv\Scripts\activate
  • Path separator is ; instead of :

macOS

  • May need to use python3 and pip3 explicitly
  • Homebrew Python recommended: brew install python@3.12
  • Add to PATH: export PATH="/usr/local/opt/python@3.12/bin:$PATH"

Linux

  • Install Python 3.12 from your distribution's package manager
  • Ubuntu/Debian: sudo apt install python3.12 python3.12-venv
  • Fedora: sudo dnf install python3.12
  • Arch: sudo pacman -S python
